# soop-gamepinball-helper

SOOP 종겜핀볼 도우미입니다.

SOOP 방송에서 별풍선 후원 메시지를 읽어 핀볼을 자동 집계하는 Electron 데스크톱 앱입니다.

종겜 핀볼 컨텐츠를 염두해두고 만들었지만, 다른 핀볼 컨텐츠에서도 활용 가능할 것 같습니다.

## 시작하기

### 의존성 설치

```bash
pnpm install
```

### 개발 실행

```bash
pnpm run dev
```

### 프로덕션 빌드

```bash
pnpm run build
```

빌드 결과물은 `release` 디렉터리에 생성됩니다.

### 코드 정리 및 검사

```bash
pnpm run format
pnpm run lint
pnpm run check
pnpm run reporter
```

## 사용 방법

1. `준비` 탭에서 SOOP ID, 진행 화면에 표시할 규칙, 핀볼 단가를 설정합니다.
2. `진행` 탭에서 시작 버튼을 눌러 SOOP 채팅 연결을 시작합니다.\
방송 화면에 핀볼 규칙을 띄우기 좋게 글자크기 조절 기능도 넣어놨습니다.\
(혹시 몰라서 하단에 인식한 별풍선 목록을 조그맣게 추가해놨습니다. 잘 작동하는지 확인용)
3. 시청자가 설정된 금액으로 후원하면서 게임 이름을 보내면 핀볼 목록에 자동 입력됩니다.\
여러 개를 한 번에 넣고 싶을 때는 `게임이름*개수` 형식으로 보내면 됩니다.
4. 핀볼 마감 시 정지 버튼을 눌러 채팅 연결을 중단합니다.
5. `검토` 탭에서 자동 입력된 핀볼을 확인하고 필요하면 직접 수정합니다.\
(우측 상단의 `인식한 별풍선을 기준으로 초기화` 버튼으로 직접 수정한 부분을 날릴 수 있습니다.)
6. `핀볼` 탭에서 `핀볼 사이트 열기` 버튼을 누르면 핀볼 사이트가 열리면서 핀볼이 자동으로 들어가 있습니다.\
(전투3 유튜브도 넣어 두었습니다.)
7. `리롤 타이머 열기` 버튼을 눌러서 타이머를 열 수 있습니다.

우측하단에 게임 플레이 타임 알아보는 사이트와 주요 게임 구매처 링크를 추가해두었습니다.

### 별풍선 인식 규칙

별풍선 이벤트와 채팅 이벤트를 같은 사용자 기준으로 연결합니다. 후원 후 30초 안에 들어온 채팅 메시지를 핀볼 입력으로 처리합니다.

예를 들어 핀볼 단가가 `50`, `100`으로 설정되어 있을 때:

- `50`개 후원 후 `게임A` 입력: `게임A` 1개 추가
- `100`개 후원 후 `게임A*2` 입력: `게임A` 2개 추가
- `100`개 후원 후 `게임B` 입력: `게임B` 1개 추가
- 설정된 단가와 맞지 않는 별풍선은 집계하지 않음

### 리롤 타이머

`핀볼` 탭에서 리롤 단가를 설정하고 리롤 타이머 창을 열 수 있습니다.

타이머가 실행 중일 때 설정된 리롤 단가와 동일한 별풍선이 들어오면 타이머가 멈추고 리롤 알림이 표시됩니다.

## 프로젝트 구조

```text
.
├─ electron/            # Electron main/preload, SOOP 채팅 연동, 저장소 설정
├─ public/              # 아이콘 등 정적 자산
├─ src/
│  ├─ components/       # UI 컴포넌트
│  ├─ helper/           # 후원 메시지 계산 로직
│  ├─ hooks/            # renderer 훅
│  ├─ shared/           # 공유 리소스
│  └─ utils/            # 범용 유틸리티
├─ dist/                # 렌더러 빌드 결과물
├─ dist-electron/       # Electron 빌드 결과물
└─ release/             # 배포 산출물
```
